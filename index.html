About the Project

Quiet Voice AI is an innovative sign language interface designed to bridge the communication gap between the deaf/hard-of-hearing community and the hearing world. Our platform converts text and speech into real-time sign language animations, making communication more accessible and inclusive for everyone.

Built with modern web technologies and a focus on user experience, this interface provides a seamless way to translate messages into sign language, supporting both English and Hindi (Hinglish) inputs.

How to Use

1

Select Language

Choose between English or Hindi from the dropdown menu

2

Type Your Message

Enter your text in the input area (max 500 characters)

3

Send & Watch

Click Send to see your message translated to sign language
